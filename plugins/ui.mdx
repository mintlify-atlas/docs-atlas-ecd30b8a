## Overview

nvim-luffy features a beautiful, customized UI with a One Piece-themed dashboard, multiple colorschemes, and smooth interface enhancements.

## alpha-nvim

[alpha-nvim](https://github.com/goolord/alpha-nvim) provides a customizable dashboard that appears when you start Neovim.

### Configuration

Located in `lua/leyland/plugins/alpha.lua:1-69`:

```lua
return {
  "goolord/alpha-nvim",
  event = "VimEnter",
  config = function()
    local alpha = require("alpha")
    local dashboard = require("alpha.themes.dashboard")

    dashboard.section.header.val = {
      -- One Piece ASCII art (Luffy)
      "⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⣀⣤⣤⠶⠶⠶⠶⠤⠤⠤⣄⣀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀",
      -- ... (full ASCII art)
    }

    dashboard.section.buttons.val = {
      dashboard.button("f", "  Find file", ":Telescope find_files <CR>"),
      dashboard.button("r", "  Recent files", ":Telescope oldfiles <CR>"),
      dashboard.button("q", "  Quit", ":qa<CR>"),
    }

    dashboard.section.footer.val = {
      "",
      '"Maybe the One Piece was the commits you made along the way."',
      "",
    }

    alpha.setup(dashboard.opts)
  end,
}
```

### The One Piece Dashboard

When you launch Neovim without a file, you'll see:

1. **Luffy ASCII Art** - A detailed ASCII art of the Straw Hat captain
2. **Inspirational Quote** - "Maybe the One Piece was the commits you made along the way."
3. **Action Buttons** - Quick access to common tasks

Press any action key or start typing to dismiss the dashboard.

## Colorschemes

nvim-luffy includes multiple high-quality colorschemes to suit different preferences and lighting conditions.

### Configuration

Located in `lua/leyland/plugins/colorscheme.lua:1-40`:

```lua
return {
  {
    "catppuccin/nvim",
    name = "catppuccin",
    priority = 1000,
  },
  {
    "folke/tokyonight.nvim",
    priority = 1000,
  },
  {
    "miikanissi/modus-themes.nvim",
    priority = 1000,
  },
  {
    "ribru17/bamboo.nvim",
    lazy = false,
    priority = 1000,
  },
  {
    "navarasu/onedark.nvim",
    priority = 1000,
  },
  {
    "rose-pine/neovim",
    name = "rose-pine",
    priority = 1000,
    config = function()
      -- Default colorscheme
      vim.cmd.colorscheme("bamboo")
    end,
  },
}
```

### Changing Colorschemes

<Steps>
  <Step title="Try a colorscheme temporarily">
    ```vim
    :colorscheme tokyonight
    ```
  </Step>
  
  <Step title="Make it permanent">
    Edit `lua/leyland/plugins/colorscheme.lua:37`:
    
    ```lua
    vim.cmd.colorscheme("your-chosen-theme")
    ```
  </Step>
  
  <Step title="Restart Neovim">
    Restart to apply the change permanently.
  </Step>
</Steps>

## snacks.nvim

[snacks.nvim](https://github.com/folke/snacks.nvim) provides smooth scrolling and other UI enhancements.

### Configuration

Located in `lua/leyland/plugins/snacks.lua:1-14`:

```lua
return {
  "folke/snacks.nvim",
  priority = 1000,
  lazy = false,
  opts = {
    scroll = {
      enabled = true,
      animate = {
        duration = { step = 10, total = 100 },
        easing = "linear",
      },
    },
  },
}
```

## nvim-web-devicons

[nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons) adds file type icons throughout the UI.

### Features

- **File type icons** - Visual indicators in file explorers
- **Plugin integration** - Works with Telescope, nvim-tree, etc.
- **Wide coverage** - Icons for hundreds of file types
- **Color coding** - Language-specific colors

### Supported Everywhere

Icons appear in:
- Telescope pickers
- nvim-tree file explorer
- Completion menus
- Buffer tabs
- Status lines

## UI Tips

<Tip>
**Performance:** All UI plugins are loaded with `priority = 1000` to ensure the interface looks great immediately on startup.
</Tip>

<Warning>
**Nerd Fonts Required:** To display icons correctly, install a [Nerd Font](https://www.nerdfonts.com/) and configure your terminal to use it.
</Warning>

<Tip>
**Dashboard Customization:** Edit the ASCII art in `alpha.lua:8-42` to personalize your dashboard with different characters or quotes.
</Tip>
