**nvim-luffy** is a modern, opinionated Neovim configuration built with lazy.nvim and focused on developer productivity. Inspired by the spirit of adventure and discovery, this configuration aims to make your coding journey as epic as finding the One Piece.

## Why nvim-luffy?

- **Modern Stack**: Built with the latest and greatest Neovim plugins
- **Developer Focused**: Optimized for web development with TypeScript, JavaScript, Python, and more
- **Opinionated Defaults**: Sensible defaults that just work out of the box
- **Fast & Lightweight**: Lazy loading ensures plugins only load when needed
- **Beautiful UI**: Clean interface with minimal statusline and beautiful colorschemes
- **Extensible**: Easy to customize and extend to fit your workflow

## What's Inside?

The configuration is structured around the `lua/leyland/` directory and includes:

- **Core Configuration**: Options, keymaps, and autocommands
- **Plugin Management**: Lazy.nvim with automatic plugin installation
- **LSP Setup**: Mason for language server management with pre-configured servers
- **Formatters**: Conform.nvim with Prettier, Stylua, Black, and more
- **UI Enhancements**: Alpha dashboard, custom winbar, and multiple colorscheme options
- **Git Tools**: Neogit and Diffview for comprehensive git workflows
- **Navigation**: Telescope, Harpoon, and nvim-tree for efficient file navigation
